Здравствуйте уважаемый пользователь.
	Операционная система Windows-64.
Сначала запускаете сервер командой в окне консоли:
	g++ -o server server.cpp -lws2_32
затем запускаете 2 клиента командой в окне консоли:
	g++ -o client client.cpp -lws2_32
Процесс ввода:
	В окне сервера вводить ничего не нужно.
	Вводятся данные в меню1 клиента 1.
	Переключение мышкой на консоль клиента 2.
	Вводятся данные в меню1 клиента 2.
	Переход на написание сообщения пункт 2 меню 1, в обоих клиентах.
	...
	Завершение в окне сервера набрать ^C.
